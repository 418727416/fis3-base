# [fis3 base](https://github.com/yanhaijing/fis3-base)
基于fis3的纯前端解决方案demo。

## 特色
此demo有如下功能特色。
- commonjs
- ECMASCript2015
- SASS&Compass
- fis components
- [template.js](https://github.com/yanhaijing/template.js)
- [data.js](https://github.com/yanhaijing/data.js)
- html，js，css组件
- 本地模拟数据
- 开发，测试发布，发布多种模式
